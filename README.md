# career path predictor

# Project Overview

   Career Path Predictor is a web-based application designed to help students and job seekers identify suitable career paths based on their skills, interests, academic performance, and career goals. The system analyzes user data and provides personalized career recommendations to support informed decision-making.
The project is being developed using Python Flask as the backend framework and MySQL as the database management system.
